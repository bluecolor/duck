# Duck

Duck is an online data masking, redaction and sensitive data discovery platform
that leverages and extends the existing capabilities of common database management systems.

## Supported Databases

- [x] Oracle
- [ ] SQLServer
- [ ] PostgreSQL
- [ ] MySQL
